# Architecture Decision Records (ADR)

Каталог архитектурных решений проекта **diaai**. Каждый ADR фиксирует значимый выбор: контекст, альтернативы, решение и последствия.

## Принципы

- **Один ADR — одно решение.** Не смешивать несвязанные темы в одном документе.
- **Нумерация:** `adr-NNN-краткое-имя.md`, порядковый номер не меняется после принятия.
- **Статусы:** `Предложено` → `Принято` → `Устарело` / `Заменено`.
- **Формат:** контекст, рассмотренные альтернативы, решение, последствия.
- **Связь с документацией:** [vision.md](../vision.md) содержит обзор ADR; детали — в файлах ADR. [data-model.md](../data-model.md) и другие документы ссылаются на ADR, не дублируя обоснование.
